npm 
			- install javaScript libraries into an app

			- 1-app npm init - initialize app to run

			- libraries = packages or modules
				- syntax = npm install [PACKAGE_NAME] --save

			- install = latest version into node_modules folder
			- --save = add package into package.json
			- require node-specific keyword
